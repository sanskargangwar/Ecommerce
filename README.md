# 🛒 AI-Powered E-Commerce Website

An AI-powered E-Commerce platform built using the MERN Stack that delivers a modern and interactive online shopping experience. The application integrates the **Web Speech API** to enable voice-based product search and navigation, allowing users to interact with the platform hands-free. Users can browse products, manage carts, place orders, and discover products using intelligent voice commands.

## 📖 Overview

This project was developed to provide a smarter shopping experience by combining traditional e-commerce functionality with voice recognition technology. The Web Speech API allows users to search products, navigate pages, and interact with the application using natural voice commands.

The platform supports secure authentication, product management, cart functionality, order processing, and responsive design for all devices.

### 🎙️ Voice Assistant Using Web Speech API

The application integrates the browser's Web Speech API to enhance user interaction.

#### Supported Voice Commands

- "Search for smartphones"
- "Show laptops"
- "Open cart"
- "Go to home page"
- "Show electronics"
- "Show fashion products"
- "Add product to cart"

#### Voice Search Workflow

1. User clicks microphone icon.
2. Speech Recognition starts.
3. User speaks a command.
4. Voice is converted into text.
5. Application processes the command.
6. Relevant products or pages are displayed.

#### Benefits

- Hands-Free Navigation
- Faster Product Discovery
- Improved Accessibility
- Enhanced User Experience
